# Color Palette Generator

A simple web app that generates a random color palette. Each palette contains five colors with their hex values displayed below each color block.

## Features

- Generate a new random palette with one button click
- Display hex values for each color
- Responsive layout for desktop and mobile screens
- Copy icons included for future clipboard functionality

## Project Structure

```text
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

Open `index.html` in your browser.

No build tools or package installation are required.

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome icons
