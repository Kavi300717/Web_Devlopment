# Case Converter

A simple case converter web app made for fun and timepass.

## About

This project converts typed text into different text cases. It is a small beginner-friendly project built with HTML, CSS, and JavaScript.

## Features

- Convert text to uppercase
- Convert text to lowercase
- Convert text to title case
- Clean and responsive UI
- Gradient background and styled buttons

## Tech Stack

- HTML
- CSS
- JavaScript

## How To Use

1. Open `index.html` in your browser.
2. Type or paste text inside the textarea.
3. Click one of the buttons:
   - `Uppercase`
   - `Lowercase`
   - `Titlecase`
